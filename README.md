# fake-profile-detection
FakeGuard: Fake Social Media Account Detection using Machine Learning
A machine learning-based system that detects fake social media accounts using user behavior, metadata, and activity patterns. The model classifies accounts as real or fake with high accuracy.
#features
- Detects fake vs real social media accounts
- Uses ML models like Random Forest and XGBoost
- Feature engineering based on user behavior
- Model evaluation using multiple metrics
- Scalable and easy to extend
